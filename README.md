# Building a VMWare Network of an Active Directory Server
![image](https://github.com/andrewkim0129/VirtualBox-ADDS-Project/assets/27276227/8fab8c95-6371-4015-a43e-d8256bcfb01b)


## Introduction

In this project, I built a home lab using Oracle VirtualBox to create a VMWare network. This network consisted of a Domain Controller functioning as an active directory server and a client machine running Windows 10. I configured NAT and routing on the domain controller server, enabling the client machine to connect to the domain within the internal network. Additionally, I set up DHCP on the domain controller server to manage the client connection
